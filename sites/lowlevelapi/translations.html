Translations

SWAC supports translations in GUI and data.

Translate HTML areas on need

When adding data over components translation is done automatically for every added dataset output.
However, if you place Elements outside of a component translation is not performed on those elements. You can start translation process by useing the translateAll() function.


// Translate contents of HTML elements
window.swac.lang.translateAll(elem);
