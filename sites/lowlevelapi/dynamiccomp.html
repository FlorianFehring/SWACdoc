Use components dynamically

Components can be build from script and dynamically added to your page.
First build the requestor tag by script and then call the component build script on that tag.
Please note: Although this is possible think twice before you use it, in most cases the placement of the requestor tag within the template or page code is more convinient.


// Build state string
let reqState = 'Present FROM my_data';

// Create requestor element
let reqElem = document.createElement('div');
reqElem.setAttribute('id','my_custom_id');
reqElem.setAttribute('swa',reqState);
// Optional: Set custom template
repElem.innerHTML = '<div>This is my awesome template written as documented within Concepts / Design</div>';
contElem.appendChild(reqElem);
                
// Build component
let viewHandler = new ViewHandler();
viewHandler.load(reqElem);
