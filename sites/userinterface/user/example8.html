Example 8: Check if user is logged in and use usersdata

You can request the user component to get information about the currently logged in user.


// Get user User component from page
let userElem = document.querySelector('[swa^="User"]');
// Check if component was found and user data is there
if(userElem?.swac_comp.userData) {
    ...
    userElem.swac_comp.userData.isadmin // usersdata should provide this boolean flag
}
