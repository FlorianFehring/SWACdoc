Example 7: How to use userdata on page

Userdata is loaded after login and stored in localstorage. To use users data on the page, create tags with a class named user_ followed by the attribute you want to display on page.


<div class="user_firstname"></div>
<div class="user_lastname"></div>
<div class="user_otherattribute"></div>
