Example 3: Use custom logo and link

By default the component searches and uses a logo file found in the /content directory. It supports the formats *.avif, *.webp and *.png. If you want to use your own logo location or filename use the option logoName which is relative to the /content directory.
By default a click on the logo redirects the user to the /index.html. You can change this behavior with the logoLink option which is relative to the pages url.
