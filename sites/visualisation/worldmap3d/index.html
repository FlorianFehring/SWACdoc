The Worldmap3d component can visualise data. It supports GLTF, GeoJson, 3DGeoJson, Tilesets and visualisation of any json data that has a latitude and longitude information in it.

Example 3: Visualising a geojson area

This example loads multiple GeoJSON files to display areas and building outlines on the 3D globe. Each datasource can define its own styling options, such as fill color, outline color, and outline width, allowing different geographic features to be highlighted independently.

Example 4: Visualising complex geojson with useage of the description component

This example shows how to display a complex GeoJSON dataset with variable extrusion heights and colors based on data properties. Each model can include a caption and detailed description, which are displayed alongside the 3D globe, allowing users to explore both spatial and attribute information interactively.

Example 6: Useing worldmap3d and search components together

This example integrates a search component with the 3D globe, enabling users to find locations or models from multiple data sources. Example 8: Useing zoomlevels and viewport requests

This example demonstrates how to load and display different models or datasets based on the camera’s zoom level and viewport. Lower zoom levels show simpler or aggregated models, while higher zoom levels load detailed models. Visualization includes multiple attributes such as suitability for photovoltaics, solar thermal, and green roofs, each with corresponding colors and legends for easy interpretation.
