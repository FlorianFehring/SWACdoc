Example 15: Zoom to a dataset

You can esily make the map zoom to a point when it's dataset is added to the map.

There are two parameters needed in the url when calling the mappage:

  • zoomToSource = the full url or name of the source, as you would use it in FROM clause.
  • zoomToSet = datasets id to zoom at.

Example url: example15.html?zoomToSource=../../../data/visualisation/worldmap2d_3d/example13/airquality_measurements.json&zoomToSet=2
