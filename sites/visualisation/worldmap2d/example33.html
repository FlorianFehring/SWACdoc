Example 33: Create new datapoints on map

With activated CreateObjectModal plugin you can create datapoints directly on the map. Click anywhere on the open space to create a new object.
Datapoint datasets are automatically created in the mainsource useing the SWAC components save() method.
Note that, because this is an static example without database, there is no new object created in real, if you reload the page the data is reset to the startpoint.

IMPORTANT

This example and maybe the plugin needs to be overworked to realise a realisation with less configuration.
