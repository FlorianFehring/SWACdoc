Example 2: Create custom form

Create easily custom forms by createing only the html form. You can use as many input forms as you wish. All input data is send as values for the keys of their input field names.

Product description 

<div id="form_example2" swa="Form">
    <form class="uk-form-stacked">
        <fieldset class="uk-fieldset">
            <legend class="uk-legend">Product description</legend>

            <div class="uk-margin">
                <label for="name" class="uk-form-label">Products name</label>
                <input name="name" id="name" class="uk-input" type="text" aria-label="Products name">
            </div>

            <div class="uk-margin">
                <label for="desc" class="uk-form-label">Product description</label>
                <textarea name="desc" id="desc" class="uk-textarea" rows="5" aria-label="Products description" required></textarea>
            </div>

            <div class="uk-margin">
                <label for="quality" class="uk-form-label">Product quality</label>
                <input name="quality" id="quality" type="range" min="0" max="100" step="1" value="50">
            </div>

            <div class="uk-margin">
                <button class="uk-button" type="submit">Send</button>
            </div>
        </fieldset>
    </form>
</div>
