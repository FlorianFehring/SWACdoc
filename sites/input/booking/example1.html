Example 1: Show available rooms

Booking component allows to list items, such as rooms, devices or other stuff available for booking. Useing a second datasource with made bookings it's possible to show the current booking state.
The bookingdata entered by the user can be send to a seperate script, allowing it to use enhanced backend scripts, like sending a verification email.
